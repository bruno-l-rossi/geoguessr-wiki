# Postes do Brasil — Identificação Regional

Os postes elétricos são uma das pistas regionais mais detalhadas disponíveis no [[brazil|Brasil]] dentro do GeoGuessr. O tipo de poste, isolador, topo e código de identificação pintado pode identificar o estado com boa precisão.

## Tipos de Poste

### Poste em Escada (Ladder Pole)
- **Aparência:** Base com segmentos retangulares horizontais escalonados + pequenos furos circulares próximos ao topo.
- **Distribuição:** O tipo mais comum em todo o Brasil.
- ⚠️ O Paraguai usa postes similares — confirmar com idioma e outros elementos.

### Poste Redondo de Concreto
- **Aparência:** Cilíndrico, liso.
- **Distribuição:** Sul do Brasil, especialmente **São Paulo**. Centros de grandes cidades em geral.

### Poste de Madeira
- **Distribuição:** Rio Grande do Sul, Roraima, Amazonas, Rio de Janeiro.

## Topos de Poste por Estado

| Configuração | Estado(s) |
|---|---|
| Braço assimétrico + suporte até o meio do lado longo | **São Paulo** (padrão), RS, RJ, MS |
| Braço assimétrico **sem** suporte | **Ceará** e **Rondônia** |
| Barra em zigue-zague sobre braço plano (poste grande) | **Ceará** e **Roraima** |
| Cone liso no topo ("pencil pole") | **Rio de Janeiro**; raramente Brasília e Manaus |
| Formato **gama (Γ)** com caixas de eletricidade | **RJ**; Belém, Salvador, Fortaleza, Maceió |
| Isoladores alternados horizontais (topo único) | **Mato Grosso** e sul do Acre |

**Suportes no braço transversal:** frequentes em geral; notavelmente raros no Nordeste.

## Isoladores por Estado

| Isolador | Estado(s) |
|---|---|
| Cogumelo **marrom** | **Pernambuco** (exclusivo) |
| Cogumelo **branco** | **Bahia**; raramente Brasília |
| **Transparentes** | **Alagoas**, **Santa Catarina**; raramente Sergipe, Ceará, Brasília |
| Forma de **carretel de linha** | **Rio de Janeiro** e **Espírito Santo** |
| **Longos e finos** | **Ceará**; raramente Goiás |
| **Horizontais** | **Pará** (muito comum), Mato Grosso |
| **Longos cinzas** (verticais) | Pará, MT, TO, BA, MS, PR, SC |
| **UFO duplo preto em tríade** | **Maranhão** |
| **Pirulito** (esférico/cônico suspenso) | **RN, Paraíba, Pernambuco**; raramente Sergipe |
| **Marrom-escuro com base cinza** | **Santa Catarina** e **Rio Grande do Sul** (quase exclusivo) |

## Códigos de Identificação Pintados/Placas

| Código | Características | Estado(s) |
|---|---|---|
| Tinta amarela longa + código com **letra inicial** | — | **Bahia e Pernambuco** |
| Tinta amarela longa + código com **número inicial** | — | **Goiás** |
| Placa pequena **amarela ou laranja** com arame | Mais alta em PE | **RN e Pernambuco** |
| Tinta **preta** + texto amarelo ou branco | — | **Paraná** |
| Placa pequena **bege** embutida na superfície | — | **Maranhão**; raramente Pará, Piauí |
| ID quadrado **amarelo** (três letras + dois números + linha + quatro números) | Letras = município | **São Paulo** |
| **Nome de rua pintado** no poste na interseção | — | **Paraná**; às vezes SC, RS |

## Lâmpadas por Cidade

| Cidade / Região | Característica Distintiva |
|---|---|
| **Rio de Janeiro** | Braço totalmente curvo do início ao fim |
| **Teresina** | Cabeça azul + braço laranja |
| **Belém** | Cabeças frequentemente laranjas |
| **Cuiabá** | Verde-claro; postes com até 6–12 medidores |
| **Belo Horizonte / Uberaba** (Gen 4) | Placa branca numerada (BH: começa 0 ou 1; Uberaba: 1 ou 2) |
| **Caxias do Sul** | Poste em zigue-zague + cabeça amarela |
| **Boa Vista / Normandia / Mucajaí** | Lâmpadas vermelhas |
| **Rorainópolis** | Lâmpadas azuis |
| **Pacaraima / Iracema** | Lâmpadas verdes |
| **Caracaraí** | Lâmpadas amarelas |
| **Ceará (geral)** | Braços chiques e coloridos; às vezes com nome da cidade |
| **Florianópolis** | Postes pintados branco + faixas azul e vermelha |
| **Norte do Paraná** (noroeste) | Lâmpadas sem braço (fixadas diretamente) |

## Outros Elementos nos Postes
- **Contadores de eletricidade:** Cuiabá, Macapá, grandes cidades do Pará.
- **Ninhos de João-de-Barro** (esferas de barro no topo): Sul do Brasil em geral.
- **Braço da lâmpada integrado** ao poste (sem placa de fixação): MT, MS, Pará, Amapá, RN, Pernambuco, SC.
