# Alfabetos e Scripts — Identificação Visual

Scripts não-latinos identificam um país ou conjunto muito pequeno de países **imediatamente**, mesmo sem ler o conteúdo. São a pista mais rápida disponível quando há texto visível.

---

## Scripts que Identificam País Único

### Japonês — Kanji + Hiragana + Katakana
[[japan]] é o **único país do GeoGuessr** com os três sistemas simultaneamente:
- **Kanji**: símbolos complexos emprestados do chinês; presentes em sinais formais
- **Hiragana**: mais simples, com loops e traços suaves — textos correntes
- **Katakana**: geométrico, angular — predomina em publicidade e estrangeirismos
- **Como diferenciar do chinês**: hiragana tem loops que o chinês não tem; katakana tem formas muito geométricas

8 kanji mais comuns em nomes de cidades: 山(montanha), 川(rio), 野(campo), 市(cidade), 町(vila), 村(aldeia), 県(prefeitura), Norte/Sul/Leste/Oeste (北南東西)

### Coreano — Hangul
[[south-korea]] — sistema de escrita com blocos silábicos distintos:
- Círculos completos (ㅇ, ㅎ, ㅗ, ㅜ) são muito característicos
- Blocos quadrados: combinação de consoantes e vogais em quadrado
- ⚠️ Não confundir com japonês — hangul não tem kanji; forma dos blocos é diferente

### Sinhala
[[sri-lanka]] — script com **círculos e curvas** pronunciados; extremamente ornamental:
- ⚠️ Exclusivo do Sri Lanka — nenhum outro país do GeoGuessr usa sinhala
- Frequentemente ao lado de texto em tâmil (outro script sul-asiático curvilíneo)

### Tailandês
[[thailand]] — script com **pequenos círculos** e espirais minúsculas no topo de muitas letras:
- Aspecto compacto e curvilíneo com detalhes circulares acima das letras
- ⚠️ Confusão com Laos ([[laos]]): Lao é similar mas com menos detalhes circulares; o stop sign lao tem **2 caracteres**, o tailandês tem **3**
- ⚠️ Tailândia dirige pela esquerda; Laos e Camboja pela direita

### Khmer (Cambojano)
[[cambodia]] — script com **hastes longas e acentos diacríticos embaixo** das letras:
- Linhas mais longas e verticais que o tailandês
- Letras com subcríticos (traços abaixo) são muito características

### Lao
[[laos]] — similar ao tailandês mas mais simples e com menos detalhes:
- Menos ornamental; letras com curvas mais suaves
- Stop sign: **2 caracteres** (tailandês tem 3)

### Georgiano
Georgia *(país no Cáucaso — não disponível no GeoGuessr; sem cobertura Street View)*

### Etíope (Amárico)
*Não em países com cobertura no GeoGuessr*

## Scripts que Identificam Grupo de Países

### Cirílico
Usado em múltiplos países — identificar qual pelos detalhes:

| País | Letras/Diacríticos Exclusivos | Outras Pistas |
|---|---|---|
| [[russia]] | Apenas cirílico padrão; "Ы" (linha dupla) comum | País grande; muitos contextos |
| [[ukraine]] | "Ї" (i com dois pontos) e "І" (i simples sem ponto de base) | Bandeira azul-amarela nos sinais |
| [[belarus]] | **"Ў"** (u com curva) e **"І"** bielorrusso — exclusivos | Apenas trekker em Minsk |
| [[mongolia]] | **"Ɵ"** e **"Ү"** — letras exclusivas do mongol | Paisagem de estepe aberta |
| [[serbia]] | Usa tanto cirílico quanto **latino** — às vezes os dois juntos | Balcãs, topografia montanhosa |
| [[north-macedonia]] | Cirílico + topônimos específicos | |
| [[bulgaria]] | Cirílico padrão; influência ortodoxa | |
| [[kazakhstan]] | Cirílico + alguns textos em latim (transição em curso) | Estepe, paisagem de Ásia Central |
| [[kyrgyzstan]] | Cirílico; contexto montanhoso | |

### Árabe
Script da direita para a esquerda; letra base muda conforme posição na palavra:

| País | Como Diferenciar |
|---|---|
| [[egypt]] | Árabe + contexto do Nilo/deserto; pirâmides |
| [[iraq]] | Árabe; cobertura muito limitada em trekkers específicos |
| [[jordan]] | Árabe + inglês; contexto do Oriente Médio |
| [[lebanon]] | Árabe + **francês e inglês** — trilíngue é pista forte; postes metálicos de malha |
| [[israel-and-the-west-bank]] | **Hebraico + árabe** — dois scripts juntos; hebraico é exclusivo |
| [[qatar]] | Árabe; picape branca desfocada; placas longas brancas com listra marrom |
| [[united-arab-emirates]] | Árabe; picape branca Gen 3 |
| [[oman]] | Árabe; paisagem árida/montanhosa |
| Arábia Saudita | *Sem cobertura oficial no GeoGuessr* |
| [[turkey]] | ⚠️ Turco usa **alfabeto LATINO** (não árabe!) — mas com "ı" (dotless i), "ş", "ğ", "ö", "ü" |

### Devanagari e Scripts Sul-Asiáticos
[[india]] usa **23 idiomas oficiais** em diferentes scripts por região:

| Script | Estado(s) | Característica Visual |
|---|---|---|
| **Devanagari** (Hindi) | Norte e centro | Linha horizontal no topo conectando letras |
| **Bengali** | Bengala Ocidental (e Bangladesh) | Curvas com linha superior; mais arredondado |
| **Tâmil** | Tamil Nadu (sul) | Curvas e círculos; similar ao sinhala |
| **Kannada** | Karnataka | Letras com arcos arredondados |
| **Telugu** | Andhra Pradesh | Curvas abertas |
| **Malayalam** | Kerala | Círculos fechados e curvas |
| **Gurmuqhi** (Punjabi) | Punjab | Letras com linha no topo + hastes para baixo |
| **Oriya** | Odisha | Semicírculos no topo |

[[nepal]] usa **Devanagari** (mesmo script do Hindi).  
[[bangladesh]] usa **Bengali**.

## Alfabeto Latino com Diacríticos

A maioria dos países europeus usa o alfabeto latino, mas com caracteres especiais que identificam o idioma:

### Pistas de Alta Confiança
| Caractere | Idioma/País |
|---|---|
| **ř** (r com hachek) | **Tcheco exclusivo** — [[czechia]] |
| **ő ű** (duplo acento agudo) | **Húngaro exclusivo** — [[hungary]] |
| **ı** (i sem ponto) + **ğ** + **ş** | **Turco** — [[turkey]] |
| **þ ð** (thorn, eth) | **Islandês** — [[iceland]] |
| **æ ø å** | **Norueguês/Dinamarquês** — [[norway]], [[denmark]] |
| **ä ö** + **å** | **Finlandês/Sueco** — [[finland]], [[sweden]] (sem å = finlandês) |
| **ý** + **é** + aspas " " | **Eslovaco** — [[slovakia]] |
| **ã â ê ô** (til/circunflexo) | **Português** — [[portugal]], [[brazil]] |
| **ñ** (eñe) | **Espanhol** (toda a América Latina + Espanha) |
| **ë** (trema) | **Albanês** — [[albania]] |
| **ş ț** (s e t com vírgula) | **Romeno** — [[romania]] (confusão com turco: romeno tem ș ț, turco tem ş ğ) |
| **č š ž** (hacheks) | **Esloveno/Croata/Sérvio-latino** — [[slovenia]], [[croatia]] |

### Como Diferenciar Espanhol de Português
Ambos usam alfabeto com poucos diacríticos:
- **Português**: "ã", "ão", "ê", "ô", "ç", "à" — mais diacríticos variados
- **Espanhol**: apenas "ñ", "á é í ó ú", "ü" — mais simples; "¿¡" (sinais invertidos)

### Scripts Únicos de Ilhas/Minorias
| Script/Idioma | País | Nota |
|---|---|---|
| **Havaiano** | [[hawaii]] | Apenas 8 consoantes; todas as palavras terminam em vogal; muito reconhecível |
| **Bislama** | [[vanuatu]] | Creole inglês com regras únicas |
| **Galés** | [[united-kingdom]] (País de Gales) | "LL", "CH", "RH" — parece inglês mas com combinações impossíveis |
| **Bretão** | [[france]] (Bretanha) | "Plou-", "Ker-", "-ec", "-ig" nos topônimos |
| **Corso** | [[france]] (Córsega) | Topônimos terminando em -u, -iu, -ju |

## Idiomas com Palavras-Chave Diagnósticas

Mesmo sem reconhecer o script completo, certas palavras são diagnósticas:

| Palavra | Significado | País/Idioma |
|---|---|---|
| **"ALTO"** | STOP | [[mexico]], alguns países da América Central |
| **"PARE"** | STOP | [[brazil]] e alguns países lusófonos |
| **"DUR"** | STOP | [[turkey]] |
| **"ARRÊT"** | STOP | Québec, [[canada]] (francófono) |
| **"Rue"** | Rua | [[france]] e países francófonos |
| **"-gade"** | Rua (dinamarquês) | [[us-virgin-islands]] — herança colonial dinamarquesa |
| **"karrika"** | Rua (basco) | [[france]] — País Basco |
| **"-gasse/-straße"** | Rua (alemão) | [[france]] Alsácia ou [[germany]] |
| **"vul"** (вул) | Rua (bielorrusso) | [[belarus]] |
| **"Safaricom"** | Operadora de telecom | [[kenya]] — exclusivo |
