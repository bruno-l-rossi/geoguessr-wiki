como configurar
1
Crie uma conta gratuita no Claude.ai
O GeoScout roda dentro do Claude.ai — você vai precisar de uma conta para usar.
~1 min
🌐
Acesse claude.ai e clique em "Sign up". Você pode criar conta com Google, Apple ou e-mail — é gratuito.
✅
O plano gratuito é suficiente para usar o GeoScout normalmente.
2
Crie um novo Project chamado "GeoScout"
Os Projects guardam o contexto do wiki permanentemente — sem precisar colar nada a cada uso.
~1 min
📁
Na barra lateral esquerda, clique em "Projects" → "New Project". Nomeie como GeoScout.
⚙️
Clique em "Set project instructions" e cole o prompt abaixo — ele transforma o Claude no agente GeoScout.
Prompt do agente — copie e cole nas instruções do Project
Você é o GeoScout, um especialista em GeoGuessr. Sua ÚNICA fonte de conhecimento são os arquivos do wiki anexados a este projeto — não use conhecimento externo. Quando o usuário descrever o que vê na tela do GeoGuessr, responda SEMPRE neste formato: 🌍 **País mais provável:** [Nome] — confiança: alta / média / baixa **Por quê:** [quais pistas do wiki confirmam essa escolha] **Alternativas:** [2-3 outros candidatos e o que os diferencia] **Para confirmar:** [1-2 pistas adicionais que resolveriam a dúvida] Se as pistas forem insuficientes, pergunte o que mais o usuário pode observar. Responda sempre em português.
3
Baixe e importe a base de conhecimento
O wiki com +140 países é o cérebro do GeoScout — você vai fazer upload uma única vez.
~3 min
📦 Wiki GeoScout — base completa
+140 países · conceitos · guias gerais · ~15MB
⬇️ Baixar wiki
📂
Após baixar, descompacte o arquivo. Dentro da pasta, navegue até geoguessr-wiki-main/wiki/.
📤
No seu Project no Claude.ai, clique em "Add content" → "Upload files". Selecione todos os arquivos .md das pastas countries/, concepts/ e o arquivo index.md.
⏳
Aguarde o upload terminar. O Claude.ai vai processar os arquivos automaticamente — isso pode levar alguns segundos.
4
Comece a usar!
Abra uma nova conversa dentro do Project e descreva o que você vê na tela do GeoGuessr.
pronto 🎉
💬
Dentro do Project GeoScout, clique em "New chat" e descreva o que você vê: vegetação, placas, arquitetura, postes, tipo de câmera, idioma, clima — qualquer detalhe ajuda.
🔁
A cada nova partida, basta abrir um novo chat dentro do mesmo Project — o wiki já está lá, não precisa configurar nada de novo.
💡
Dica: quanto mais detalhes você descrever, mais precisa será a análise. Tente mencionar pelo menos 3 pistas diferentes.
